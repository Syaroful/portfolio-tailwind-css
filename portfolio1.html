Hotelio:
A Hotel Booking Mobile App

Project Description:

Hotelio is a mobile-based hotel booking application that I developed as part of the Fullstack Flutter Developer course at BuildWithAngga.com. This project aims to understand the process of creating a hotel booking application, starting from app design, page slicing, to integration with the Firebase database.

Publication Link:

Project Link

Philosopy or Concept:

The main concept behind Hotelio is to provide an intuitive and efficient hotel booking experience for users. The focus is on an attractive design and functionality that enables users to easily find and book accommodations according to their needs.

Project Objective:

The goal of Hotelio is to practically learn how to develop a comprehensive hotel booking application using Flutter and Firebase. I aimed to gain a deep understanding of the workflow of mobile app development from design to database implementation.

Development Process:

Creating Hotelio involved steps in UI/UX design using Figma, using Dart language in the Flutter framework for implementation, and integration with the Firebase database. I utilized Visual Studio Code as the development environment for writing code and testing the application.

Skills Utilized:

In this project, I applied skills in mobile app development with Flutter, used Firebase for database management, employed Figma for UI/UX design, and utilized proficiency in the Dart programming language.

Technologies or Tools Used:

  • Dart programming
  • Flutter framework
  • Firebase database
  • Visual Studio Code (IDE for development)
  • Figma (for UI/UX design)
